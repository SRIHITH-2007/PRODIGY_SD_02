# PRODIGY_SD_02
A simple number guessing game where the program generates a random number between 1 and 100. 
The user keeps guessing until they get it right.

Features
> Random number generation
> User input handling
> “Too high / Too low” hints
> Attempt counter

How to run ?
        javac GuessingGame.java
        java GuessingGame

Sample Output
       Guess the number (1 to 100):
       Enter your guess: 50
       Too high! Try again.
       Enter your guess: 30
       Too low! Try again.
       Enter your guess: 40
       Correct! You guessed the number in 3 attempts.
